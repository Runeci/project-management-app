О ПРОЕКТЕ

Возможности проекта

Приложение является прототипом Trello, разработано с использованием Angular, RxJS. Для хранения и обработки необходимых данных обеспечено взаимодействие с сервером.

Дизайн и интерфейс приложения

Все страницы приложения оформлены в едином стиле, простой и интуитифный дизайн, плавные анимации, применены стили для интерактивных элементов, появляющийся pop-up с пояснением при ошибках сети, при попытке удаления элементов реализован confirmation modal с подтверждением серёзности намерений.

Основная функциональность

Для работы над проектом используются доски - пространство, где можно следить за информацией. Доска состоит из колонок прогресса. На доску можно добавить любое количество колонок, а затем упорядочить и переименовать их по своему вкусу. В каждой колонке можно создать карточку с заданием или идеей. Все поля можно редактировать или удалить. Перетаскивание карточек или перемещение колонок осуществляется с помощью drag-n-drop.

Адаптивная верстка

Поддерживаются разрешения для десктопа (1024px), планшетов (768px) и мобильных устройств (не менее 320px).
